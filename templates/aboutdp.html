Epsilon Visualizer + Optimizer

This tool both visualizes how epsilon works for users and calculates the optimal epsilon value that maximizes the welfare function for differential privacy parameters.

Meet the Team

This project was created by Vihaan Gupta, Jade Nair, Julia Shephard, and Natalia Siwek for Harvard's CS208 class.

Codebase

The code for this project can be found on GitHub at https://github.com/jnair37/cs208-tool.
